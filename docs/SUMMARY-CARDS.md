  -font "JetBrains-Mono" -pointsize 24 \

## Usage Guidelines

1. **Always include:**
   - PSA branding
   - GitHub link
   - Clear call-to-action

2. **Color scheme:**
   - Primary: Catppuccin Mocha (`#1e1e2e` background, `#cdd6f4` text)
   - Accent: `#cba6f7` (purple), `#89b4fa` (blue), `#a6e3a1` (green)

3. **Typography:**
   - Code: JetBrains Mono, Fira Code, or Hack Nerd Font
   - Headers: Bold, size 24-32pt
   - Body: Regular, size 14-18pt

4. **Icons:**
   - Use Nerd Font icons when possible
   - Emoji for accessibility
   - Keep consistent style
